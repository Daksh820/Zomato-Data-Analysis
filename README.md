# 🍽️ Zomato Restaurant Analysis

## 📌 Project Overview
This project analyzes the Zomato restaurant dataset using Python, MySQL, and Power BI. The project covers data cleaning, SQL analysis, and dashboard creation to generate meaningful business insights.

## 🛠️ Tools Used
- Python (Pandas)
- MySQL
- Power BI

## 📂 Project Workflow
1. Data Cleaning using Pandas
2. Missing Value Handling
3. Duplicate Removal
4. Business Logic Validation
5. SQL Analysis
6. Power BI Dashboard

## 📊 Dashboard KPIs
- Total Restaurants
- Average Rating
- Average Cost for Two
- Total Votes

## 📈 Dashboard Visuals
- Top 10 Cities
- Rating Distribution
- Online Delivery Analysis
- Table Booking Analysis
- Price Range Analysis
- Cuisine Analysis
- Restaurant Details Table

## 🔍 Key Insights
- Identified the top 10 cities with the highest number of restaurants.
- Analyzed restaurant ratings and customer votes.
- Compared restaurants offering online delivery and table booking.
- Studied pricing patterns across different price ranges.
- Built an interactive dashboard for business decision-making.
